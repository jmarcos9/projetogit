Método de Ordenação Selection Sort


Funcionamento:

É um método de ordenação por seleção

Consiste em ordenar a lista “selecionando” a cada iteração o menores itens possíveis e os colocam da esquerda para a direita.

Para cada elemento é necessário pecorrer o vetor a partir do elemento atual, necessitando para isso de dois loços: um para cada elemento da lista e outro para cada um desses elementos percorrer toda a lista.


Representação Gráfica Animação Vídeo

Vantagens:

  • Fácil para implementar
  • Não necessita de um vetor auxiliar
  • Ocupa menos memória
  • Boa performance em vetores pequenos

Desvantagens:

  • Lento em vetores grandes
  • Algoritmo não é estável (não preserva a ordem relativa dos elementos iguais)
  • Faz sempre a mesma quantidade de comparações, mesmo que o do vetor já esteja ordenado
